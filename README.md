# Daily Reflection Decision Tree

## Approach
Designed a deterministic decision tree to map structured inputs to fixed outputs.

## Inputs
- Task completion
- Mood (1–5)
- Distractions

## Output
- Reflection
- Suggestion

## Determinism
Same inputs always produce same outputs.

## AI Usage
AI used only for parsing natural language into structured format.

## Guardrails
- JSON schema enforcement
- No assumption policy
- Input validation
